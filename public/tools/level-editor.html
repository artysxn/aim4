Create

Add multiple spawns per team, the game picks one at random on spawn and respawn. Each team needs at least one spawn.

Transform

Click to select. Shift-click to add/remove. Scale freely; snaps to the nearest grid step on release. R 45° · Shift+R 90° · [ / ] slope tilt.

Selected

Nothing selected, click an object in the view.

Level

Bounds: the red box

The red wireframe box is the arena bounds: the outer wall of the playable area. Players and the camera stay inside it. Auto wraps it around your boxes + spawns with a margin; Manual sets exact limits.

Export / Import

Format matches in-game maps. Includes boxes, clipBarriers, and spawnZones. Spawns: A / B are one point each, or an array for random pick.
